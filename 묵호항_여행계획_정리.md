## 4. 핵심 제약조건 / 리스크 플래그

1. **"20시의 벽"** — 문어닭강정 일요일 20:00 마감. 장보기를 1차 후에 하기로 했으므로 **저녁 1차는 19:30까지 반드시 종료**해야 닭강정 픽업 가능. 어판장 회는 21:00 마감이라 여유 있음.
2. **닭강정 주말 품절** — 8/16이 일요일이라 품절 리스크 높음. 사전 전화 예약 필수급.
3. **어달해변포차 = 노지** — 우천 시 운영 불투명. 당일 날씨 확인 + 실내 대안 필요.
4. **월요일 휴무 주의 (2일차)** — 무코문어 1936, 오뚜기칼국수는 월요일 휴무. 2일차 대안으로 쓸 수 없음.
5. **전원 음주** — 렌트 시 첫날 저녁부터 운전 불가. 밤 이동은 택시 필요 (펜션↔어달 5분 거리라 부담 적음).
6. **체크인/아웃 시각 미확정** — 타임테이블 앞뒤 기준점이므로 펜션에 확인 필요.

## 6. 의사결정 로그 (경위)

- 점심/저녁 후보 8곳 검토: 묵호항닭 문어닭강정 / 비비소 / 무코문어 1936 / 장수돌짜장 / 거동탕수육 / 고향의맛손칼국수강자옹심이 / 오뚜기칼국수 / 초당쫄면순두부
  - → 거동탕수육 = 1일차 점심 후보, 비비소 = 2일차 점심 확정, 닭강정 = 3차 장보기
  - → 고향의맛옹심이는 일요일 휴무로 1일차 불가 / 장수돌짜장은 위치(어달 쪽)·평점 애매 / 무코문어는 1차 후보로 제안됐으나 미확정
- 장보기 타이밍: "체크인 직후" → **"저녁 1차 종료 후"로 최종 변경** (사용자 결정)
- 놀거리: Claude가 논골담길/묵호등대/도째비골 제안 → 사용자가 **한섬해수욕장 단독**으로 확정
- 귀가 열차: 846(19:11)은 예약대기로 제외 → 886(15:48) vs 888(16:52) 비교 → **886 확정**
- 3차 메뉴: 문어닭강정 + 회 (숙소에서)
- 산출물: xlsx로 먼저 제작 → **HTML 올인원 페이지로 방향 전환** → 다꾸(다이어리 꾸미기) 감성으로 리디자인 → 가독성 위해 폰트 교체 (제목 주아체/나눔손글씨펜, 본문 고운돋움)
